Bienvenido al: ¡Proyecto JavaScript #2

Para este ejercicio estarás haciendo una aplicación web muy sencilla que contiene un arreglo con nombres "de alumnos" en un archivo .js. El sitio web le pedirá tres comandos posibles: agregar, mostrar y salir. Esta página web solicitará al usuario uno de los comandos. Esto es lo que cada comando debe hacer:

  • agregar: Muestra un prompt que solicita un nombre de estudiante estudiante. A continuación, agregua este nombre de estudiante al arreglo de nombres de los estudiantes en el archivo .js
  • mostrar: Imprimirá la lista de estudiantes usando console.log
  • salir:Finalizará el bucle while de prompts.
